MOSS Radio

My first Ruby On Rails application, working with an awesome team of 3 others for our Le Wagon bootcamp final project.

The prototype/idea of MOSS is a community radio service using AJAX and minimal front-end styling for clean navigation and minimal page reloads. AJAX (Asynchronous JavaScript and XML) allows the central window to load content dynamically without requiring a full page reload to give a smooth user experience.
